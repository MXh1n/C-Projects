# My C Programming Journey

A collection of C projects built while learning programming from the ground up. Each project represents a milestone in understanding computer science fundamentals.

## 📚 About This Repository

This repository documents my journey through C langauge. Each project focuses on applying specific concepts in practical, working programs.

### Why C?
- Learn how computers actually work at a low level
- Understand memory management and pointers
- Build a strong foundation for other languages
- Appreciate what higher-level languages abstract away

## 🛠️ Development Environment
OS: Ubuntu linux
Editor: Vim
Compiler: GCC
